Email Security Analyzer
A machine learning web application that detects malicious emails and categorizes them based on user interests.

Features
Email Classification: Identifies malicious vs safe emails using ML

Custom Categories: Monitor emails related to:

Sports (basketball, football, cricket, tennis)

Offers & Deals (discount, sale, promotion)

E-shopping (Amazon, Flipkart, eBay)

Job Alerts (vacancy, job postings, TCS, Wipro)

Dual Modes:

TEST Mode: Analyze single or multiple emails

INBOX Mode: Bulk analyze emails from CSV files

Real-time Analysis: Instant results with visual indicators

Export Results: Download analysis as CSV files
